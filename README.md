### 介绍
哈尔滨学院21级毕业设计，基于uni-app的溯源平台项目

### 安装与运行
确保自己的电脑拥有 node 环境且拥有 HBuilderX 和 微信开发者工具，以下是开发时依赖的版本信息：

- node：v18.15.0
- WeChatLib: 3.5.3

### 目录结构
- page ( 小程序页面 )
- static ( 静态资源 )
- wxcomponents ( 微信小程序组件 )
- manifest.json ( uni-app配置 )
- pages.json ( 小程序页面配置 )
